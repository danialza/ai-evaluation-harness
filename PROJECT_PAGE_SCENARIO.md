# Website Project Scenario

## Project title

AI Evaluation Harness From Scratch

## One-line summary

A tiny Python project that shows how AI models are tested with questions, prompts, model answers, grading, and a final score.

## The story

I wanted to understand AI evaluation without hiding behind big frameworks.

So I built the smallest useful version first:

- a tiny dataset
- a prompt template
- a model connector
- a grader
- a JSON result file

Then I added a slightly more realistic version with LangChain and Gemini.

## Demo flow for the page

1. Show the simple diagram: Dataset -> Prompt -> Model -> Grader -> Score.
2. Show the beginner script running locally.
3. Show the generated `results.json`.
4. Show the optional Gemini command.
5. Explain what I learned in plain English.

## Suggested screenshots

## Tech stack

- Python
- LangChain
- Gemini API
- JSON
- GitHub

## What this project proves

- I understand the basic evaluation loop behind AI model testing.
- I can build a simple version before reaching for bigger tools.
- I can connect the same evaluation idea to a real model provider.
- I can turn model outputs into structured results for a website or dashboard.
